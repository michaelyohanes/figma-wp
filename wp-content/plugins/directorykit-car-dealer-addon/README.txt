== Description ==

Directorykit Car Dealer Addon is an addon for the WP Directory Kit plugin that allows you to launch a professional car sales portal quickly. With advanced search, filtering, and listing management, you can showcase vehicles, dealers, and automotive services with ease.

It also supports multi-vertical directories, so you can expand to other niches like real estate, jobs, pets, and local services if needed.

## Automotive Directory Features

- New and Used Car Listings
- Car Dealer Profiles
- Vehicle Comparison Tables
- Advanced Search & Filtering
- Booking Test Drives / Contact Forms
- Multi-language & RTL support
- WooCommerce integration for paid listings
- Elementor-ready templates and widgets
- Free OpenStreetMap integration for locations

== Installation ==

1. Extract `directorykit-car-dealer-addon.zip` into `/wp-content/plugins/`
2. Activate the plugin via the WordPress 'Plugins' menu
3. Install and activate WP Directory Kit plugin (required)
4. Configure your directory settings
5. Import automotive demo content for a ready-to-use portal

== Frequently Asked Questions ==

= Is this a standalone plugin? =
No. It is an addon for WP Directory Kit. It sets up fields, categories, and demo content for automotive directories.

= Can I use it for other directory types? =
Yes. While optimized for cars, it can also support real estate, jobs, pets, and other niches.

= Does it include demo content? =
Yes. Demo content for vehicles, dealers, and automotive services is included for quick setup.
