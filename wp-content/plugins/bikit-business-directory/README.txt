Bikit Business Directory WordPress Plugin with innovative features, easy customization, and full compatibility with WP Directory Kit premium tools.

== Description ==

Bikit Business Directory is a comprehensive WordPress plugin designed for modern directory websites across multiple niches. Use it to build real estate portals, job boards, pet listings, automotive directories, local places guides, classifieds, and more—with advanced search, filtering, and management capabilities.

== Installation ==

This section describes how to install the plugin and get it working.

1. Extract `bikit-business-directory.zip` into `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Install and activate WP Directory Kit plugin (required)
4. Configure your directory settings
5. Import demo content for quick setup

== Frequently Asked Questions ==

= Is this a standalone plugin? =

No. This plugin is an addon for WP Directory Kit. It configures fields, categories, and imports demo listings for your directory inside WP Directory Kit.

= Does WP Directory Kit work with all themes? =

Yes, all themes are supported.

= Which directory types are supported? =

Real estate, jobs, pets, cars/automotive, local places, services, classifieds, and more. You can tailor fields and categories per niche.

= Does the plugin include demo content? =

Yes. Comprehensive demo content is available to help you get started quickly and showcase capabilities per niche.
